# 4.0 网络层：数据平面

## 章节概述

网络层数据平面负责路由器中数据包的实际转发功能。本章严格遵循《计算机网络：自顶向下方法》第4章内容，专注于数据平面的转发机制、IP协议和相关技术，为理解网络层的基础功能提供全面的知识体系。

**注意**：本章仅涵盖网络层的**数据平面**功能。控制平面（路由算法、路由协议等）将在第5章单独讨论。
